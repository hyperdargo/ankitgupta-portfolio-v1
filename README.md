## 🚀 Getting Started Locally

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/hyperdargo/ankitgupta-portfolio-v1.git
   cd ankitgupta-portfolio-v1
   ```

2. Open `index.html` in your browser:

   ```bash
   open index.html
   ```
